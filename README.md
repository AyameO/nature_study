### 植物の全体的な知識
* 植物が緑色なのは緑色の光を反射しているから。
* 樹高 - 木の高さ
* 樹冠 - 木を上から見たときに見える葉っぱ
* 葉柄 - 葉っぱと茎の間の部分(葉柄がない種類もある)
* 葉身 - 葉っぱの葉の部分
* 葉痕 - 植物が落葉したとき茎の表面に残される傷跡のこと。葉印(よういん)ともいう。
* 羽状複葉 - 葉柄から続く葉軸の左右に小葉をつけるような配列の複葉。
* 掌状複葉 - 複葉のうち，小葉が1ヵ所から数個の方向に向って指のように分れ出ているもの
